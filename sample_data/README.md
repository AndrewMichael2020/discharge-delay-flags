# Sample synthetic data

This folder contains the curated one-large-hospital synthetic outputs used by the README.

Each scenario contains generated Parquet/CSV/JSON/HTML artifacts from the workflow:

- `balanced/`
- `diagnostics_heavy/`
- `weekend_flow_gap/`

Scale:

- 1 synthetic hospital
- 90 days
- 110 encounters per day
- 9,900 encounters per scenario

The data is fully synthetic and contains no PHI.
